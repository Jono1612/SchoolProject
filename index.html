Clicker

How to Play

1: Start the game by tapping/clicking on the screen
2:Adjust sound to comfortable level
3:Tap the button and watch your score go up
4:Make a top highscore

Description

The game is a simple easy going clicker game where everything about the button changes when you click on it. It is all for fun although it could be used in a competitive way because of the scoreboard. There is no issue in any country with this game because it's simple and very easy to play with no connections to anything but perhaps a magic button. Enjoy this game and have fun.
